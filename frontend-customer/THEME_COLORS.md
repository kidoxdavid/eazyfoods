# Theme Colors

The customer frontend uses:
- **Green (Primary)**: #16a34a (primary-600) for buttons, links, and accents
- **Nude (Secondary)**: #a08f7a (nude-600) for secondary elements
- **Black Footer**: Unchanged

If colors appear blue, try:
1. Hard refresh browser (Cmd+Shift+R on Mac, Ctrl+Shift+R on Windows)
2. Clear browser cache
3. Restart dev server: `npm run dev`
